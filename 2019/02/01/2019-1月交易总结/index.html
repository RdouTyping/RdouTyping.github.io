2019 1月交易总结

发表于 2019-02-01 | 更新于 2019-02-04 | 分类于 投资理财

总结

这个月收益基本为零，最多时候赚了4000多，最后一周基本全部亏回去了。这个月美股行情是特别好的，但我的策略不管是理论还是实际都差强人意，理论上最多也就赚2000多点。

这个月美股创了多项记录，纳指上涨9.74%，标普500指数上涨7.89%，这数据竟然比去年1月还要好。更想不到的是12月跌这么多，结果1月全部涨回来了！这就是市场，你永远不知道它接下来会怎样。

我的表现

收益

收益基本为0，扣除手续费等后亏损几十块，最大回撤6%左右。

###坚持交易系统

9/10

从7号算起，这个月只有一天（01/29）有人为干扰的操作，其他时候都很好的执行了交易系统。

自己操作的这天亏钱了，但这天本该是赚钱的，这一来一回大概少赚有小1000吧。这一天我进行操作是有原因的，下面我会进行反思。

交易系统本身

7/10

交易系统本身没有太大问题，理论上这个月也能盈利2000左右（没有扣除交易费）。但和这么好的大盘比起来就有些逊色了。没关系，慢慢来。

思考与教训

如果根据现有信息买反了方向怎么办？

我的系统需要当天的收盘价，但我是通过在收盘前15分钟下market on close来下进行交易的（因为在上班，手动在最后一分钟买入不太方便）。因此我需要通过分析12:45前的走势来预估收盘价范围。

有时候趋势很明显，并且收盘价在很大范围内都会得到相同的结果。因此根据收盘前15分钟信息是可以做出正确的选择。但有时候系统产生的结果对收盘价很敏感，有时候差0.1就会导致不同的结果。在这种情况下，我只能在12:45前凭感觉做出决定并下单。

按照1月的经验，这种操作上的误差并不会对收益带来过大的影响。因为系统的预测率也就60%左右，因此这种随机的偏差不会对系统收益造成致命的影响。

我在2月需要做两件事情，第一，详细记录每日交易，不仅仅要记录赚了多少/亏了多少，还要记录是买入的是否和系统预测一致以及这种情况下赚了多少/亏了多少；第二，研究下脚本，让TWS能在最后30秒或者15秒进行自动化下单。

特殊情况比如财报季，重要会议等该如何下单？

这个月唯一一次破例操作就是出重大财报的那天。出财报很容易引起大波动，所以我想根据现有信息进行一些人为操作。但这样值得吗？既然系统已经回测了这么多年，也遇到了这么多的财报，盈利不也很好吗？因此我觉得我还是不操作好。我要做的就是在这些天降低仓位，让赚钱/亏钱都不要太大，这样可能比自己去猜效果要好些。

而其他重要会议？除了非农，美联储会议，以及现在的中美会议，其他影响也都有限。很难把握，还是根据系统做比较好。

一句话，以后还是按照系统做，不要搞太多特殊情况。
