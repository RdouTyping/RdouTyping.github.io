如何坚持交易系统？

发表于 2019-02-01 | 分类于 投资理财

如果想要通过交易系统赚钱，那么必须坚持自己的交易系统。开仓、平仓、仓位管理一个都不能乱来，交易前告诉自己：按计划执行！按计划执行！按计划执行！

坚持按计划交易的好处

  1. 最大的好处就是不会亏钱，而且往往还能赚钱，赚很多钱（有个前提，你的系统是正期望的系统）。
  2. 轻松，压力小，睡觉睡得好，上班也不会提心吊胆、分心。
  3. 能多花时间在其他美好的事物上，比如家人。
  4. 有更多时间用来提高自己。
  5. 养成良好习惯，有助于今后执行其他交易计划。

为什么做不到？

做不到的原因很多，最主要还是贪婪和恐惧：

  1. 不愿意承认失败，贪婪，想搏一把。比如早上亏钱了，想通过日内交易赚回一部分或者全部钱；如果赚钱了，但比计划赚的少，也想搏一搏把钱赚回来。
  2. 觉得早上交易结束太快，盘中有大好机会可以抄底或者做空，不抓住就亏大了。
  3. 交易结束的盘后/或者交易没开始的盘前出现一些小行情，想多买点。
  4. 通过运气盈利几次后，觉得系统不如自己牛逼。
  5. 害怕了，提前结束交易（盘前/盘后清仓）。

怎么办？

既然按照计划交易这么好，那么我要怎么做到坚持呢？虽然这很难，但必须要克服，恐怕这已经是我第N（N>10）次这么说了吧，既然到了2019，我希望能彻底改变下。

关于看盘

看盘是最影响操作和心情的东西，如果我要坚持我的系统，我必须减少看盘次数（包括股指期货），并主动放弃盘中/前/后的一切波动。我知道改变很难，先一步一步的来：

  1. 删除雪球/老虎，新闻通过华尔街见闻和Marketwatch等查看，看盘次数不限制，但尽量少，坚持1个月。
  2. 将看盘次数减少到平均1小时一次，坚持1个月。
  3. 将看盘次数减少到平均3小时一次，坚持3个月。
  4. 将看盘次数减少到买入前、买入后、卖出后各一次，其他时间只能看3次，坚持3个月。
  5. 每小时只能看盘一次（不可累加），坚持1个月。
  6. 每2小时只能看盘一次（不可累加），坚持2个月。
  7. 每天看3次 - 开盘前、收盘前、睡觉前。

关于交易

如何保证不进行额外的交易？我不能删除IB，因为删除IB再安装会导致重新验证，很麻烦。我觉得必须通过一些其他手段增加交易难度。

  1. 每次交易完登出IB。
  2. 设置部分时间屏蔽IB？

关于心态

心态主要体现在两个方面，恐惧和贪婪以及失落。

  1. 按计划操作，从根本上减少过大失误。
  2. 减少看盘次数（尤其是盘中）。
  3. 如果开盘是亏钱，不要郁闷、失落，告诉自己亏损是系统的一部分。

奖励机制

我觉得我需要一些奖励才能让自己更好的坚持。

  1. 如果当天亏损，并且没有乱操作 + 2分。
  2. 如果当天赚钱，并且没有乱操作 + 1分。
  3. 如果当天有乱操作，-1分
  4. 连续5天都是加分的话额外加2分，第6天清零。
  5. 一周如果5天交易5分算及格，6分良好，7分以上优秀。
  6. 如果连续20天（一个月交易日）都是加分，额外奖励。

奖励项目：

一周得分6分：$10

一周得分7分及以上：$25

连续20天都是加分：额外$25

这部分需要根据实际情况修改，先暂时运行一个月，以后在适当调整。
